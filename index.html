Parfum Campaigns

Independent editorial archive of fragrance advertising campaigns and brand visuals.

This page documents the standards, structure, and conventions for the Parfum Campaigns site. It is intended to be the single source of truth for layout, images, SEO, and page composition.

Overview

Parfum Campaigns is an independent publisher and archive focused on documenting fragrance campaign imagery across major houses and historical periods. The site prioritizes clarity, consistency, and search visibility through disciplined file naming and descriptive alt text.[1]

“Grid-first. White background. No banners. Let the campaign imagery carry the page.”

Mission

The mission is to build a clean, navigable archive of fragrance campaign visuals that remains useful over time: brand-by-brand pages, short contextual descriptors, and image grids that privilege the work itself over decoration.

Scope

Included

  • Brand campaign images (print ads, hero visuals, editorial campaign stills)
  • Brand sections with headers + short descriptors
  • Two-up image grids with consistent spacing
  • Images linked to the brand’s official site (or official campaign landing pages)

Excluded

  • Hero banners and large decorative headers
  • Overdesigned backgrounds or heavy theme graphics
  • Affiliate framing, “best-of” listicles, or influencer-style content

Site structure

The site is organized as a set of brand sections. Each brand section contains:

Element Requirement Purpose
Brand header Brand name (all caps or title case), followed by a short descriptor line Scanability and consistent visual hierarchy
Two-up grid Two images per row on desktop, single column on mobile Fast browsing without clutter
Image link Each image links to the brand’s official site Authority signal + user utility
Alt text Descriptive, non-decorative, factual Accessibility + image search relevance
Optional: Single-page vs multi-page

This wiki page is single-file by design. You can keep Parfum Campaigns as a single long page, or split into brand pages later. If you split, the same naming and alt-text rules still apply.

Image standards

Images should be selected for clarity, recognizability, and campaign relevance. The visual system is grid-first and white-background.

Filenames

Use consistent SEO filenames in this pattern: 

brand-product-campaign-context.jpg
Example filename Meaning
armani-acqua-di-gio-campaign-print-ad.jpg Armani / Acqua di Giò / campaign / print ad
creed-bois-du-portugal-1987-vintage-bottle.jpg Creed / Bois du Portugal / 1987 / vintage bottle

Alt text

Alt text must be descriptive and factual. Avoid decorative filler or marketing language.

Good alt text Bad alt text
“Giorgio Armani Acqua di Giò campaign image with model in coastal setting” “Sexy luxury vibe photo”
“Vintage Creed Bois du Portugal bottle (1987) on neutral background” “Rare amazing bottle”

SEO standards

SEO is driven primarily by consistent image metadata (filenames + alt text) and clean, crawlable structure. Keep content straightforward and avoid excessive scripts or trackers.[2]

SEO footer (discreet)

If you use a footer, keep it minimal: one short line describing the archive purpose, plus any required attribution. Do not add tracking scripts or analytics.

Style guide

The site should read like an archive: restrained, factual, and consistent.

Rule Standard
Background White
Layout Grid-first, no hero banners
Copy tone Short descriptors; avoid hype
Links Images link to official brand sites
